2) CADA VISITA GUARDA SU HORA
═══════════════════════════════════════
   |------------------------------------|
   | ⬆  1  🏪 MODERN CUTS BARBERSHOP    |
   |        🕐 1:18 PM  ✏️              |
   |    🗺  [ ✓ Visitada ]  ✕           |
   |------------------------------------|

Al marcar "Visitada" guarda la hora sola.
Y el lapiz ✏️ te deja corregirla si se te olvido marcar.

Las visitas viejas (sin hora) se siguen viendo igual,
no se pierde nada.

3) EL RECORDATORIO
═══════════════════════════════════════
A los 20 MINUTOS de llegar a una barberia, si todavia
no la marcaste, sale esto:

   +----------------------------------+
   |              🕐                  |
   |   Ya terminaste en               |
   |   MODERN CUTS BARBERSHOP?        |
   |   Llegaste hace 20 minutos       |
   |                                  |
   |  [ ✓ Si, marcar visitada ]       |
   |  [ ⏰ Todavia no                 |
   |     Recuerdame en 10 minutos ]   |
   +----------------------------------+

La app sabe que llegaste cuando registras la primera
venta o pedido a un barbero de esa barberia.

Si tocas "Todavia no", te lo recuerda en 10 minutos.
Si la marcas, ya no te molesta mas.
